## 🎮 조작 방법 (How to Play)

| 키      | 기능 (한국어)           | 기능 (English)             |
|---------|-------------------------|-----------------------------|
| ↑       | 위로 이동               | Move Up                    |
| ↓       | 아래로 이동             | Move Down                  |
| ←       | 왼쪽으로 이동           | Move Left                  |
| →       | 오른쪽으로 이동         | Move Right                 |
| SPACE   | 상호작용 / 공격         | Interaction / Attack       |
| ESC     | 창 닫기                 | Close Window               |
| P       | 무기 꺼내기 / 넣기      | Equip / Unequip Weapon     |
| C       | 백스텝 (회피)           | Backstep (Dodge)           |
